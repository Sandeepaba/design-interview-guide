## вҡҷпёҸ Installation

Clone the project

```bash
git clone https://github.com/Sandeepaba/design-interview-guide.git
```

Go inside the folder

```bash
cd design-interview-guide
```

Install packages

```bash
npm install
```

Run project

```bash
npm run dev
```

Build Project

```bash
npm run build
```
